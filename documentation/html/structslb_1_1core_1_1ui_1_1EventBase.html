Detailed Description

template<class Node = SLB_NODE>
struct slb::core::ui::EventBase< Node >

The base class for the events generated by the search algorithm. Note that an event carries with it the information about its visualization as well. This information is the basis for building a visual event (VisualEvent) out of it.

Template Parameters
NodeThe node type. An event usually carries data from a search node. This data is used to display the event in the log window.
