const Node *  n,
const Node *  parentSubstitution = nullptr 
)

The function that algorithms use for logging an event. The function uses tag dispatch on the type of the log to make sure that no run-time overhead is incurred when no log is used (i.e. when the type of the log is Nothing).

Template Parameters
EventThe particular event class. It should be a class derived from EventBase.
LogThe type of the log. It should be either an instantiation of AlgorithmLog or Nothing.
NodeThe node type.
Parameters
mylogThe log.
nThe current node in the search.
parentSubstitutionThe node whose state should be used as a parent state instead of the parent stored in n. See how the ext::event::NothingToDo event is generated in ext::algorithm::Astar for an example of when this is useful.
