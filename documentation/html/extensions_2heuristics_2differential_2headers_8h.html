enum IndexKind
strong

Kinds of indexing functions.

Note
All the members of an index are static. An index without an inverse provides only the static constexpr member kind of the IndexKind type, the to function to compute an index based on step and a size member showing the size of the range of possible indices. An index with inverse provides in addition the from function to compute state based on index.
