template<class MyAlgorithm, bool = false>
struct slb::ext::policy::backtrackLock::Inplace< MyAlgorithm, bool >

Handles RAII for backtracking in IDA* when the state is updated in place.

Template Parameters
MyAlgorithmThe search algorithm.
Note
The second template parameter is true when the log of algorithmic events is kept (which necessitates storing the parent state) and false otherwise.
