template<class Node = SLB_NODE>
struct slb::core::ui::EventBase< Node >::EdgeChange

Description of change of visual representation of an edge. An edge is represented here by a pair of smart pointers to states. Note that this is different from a visual event (see VisualEvent::EdgeChange), in which an edge is represented here by a pair of vertices of the domain graph.
