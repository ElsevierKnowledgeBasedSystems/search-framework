Detailed Description

A GroupLock object is created in order that the following drawing is done in a group, possibly on top of the current drawing.
It cares for a lot of nitty-gritty details without which strange things happen.
Uses PatternLock to implement its functionality.

Warning
I got this right after a lot of trial and error. Do not try to modify this unless you are a graphics person who knows what he is doing.
