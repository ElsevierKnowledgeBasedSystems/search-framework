struct IdAstar< ALG_TPARAMS, BacktrackLock >

The IDA* search algorithm.

Template Parameters
logFlagIf true, the events generated by the search algorithm are logged. Otherwise, they are not.
Node_The search node type.
GoalHandlerThe policy for handling goal conditions.
InitialHeuristicThe initial heuristic used by the search algorithm.
