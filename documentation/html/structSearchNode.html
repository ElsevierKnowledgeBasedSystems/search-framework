template<class State_ = SLB_STATE, class NodeData_ = SLB_NODE_DATA, typename BucketPosition = SLB_BUCKET_POSITION_TYPE>
struct SearchNode< State_, NodeData_, BucketPosition >

The search node type. Inherits the data structure to make data (e.g. g- and f-value) easy to access.

Template Parameters
