int n_
 The stage. Stages are numbered from right to left: k, 0, 1, ..., k-1. The goal is at stage k in the top row (in contrast to the book, where the goal is in the bottom row). The start is at stage k-1 in the bottom row. A node in the top row has the same number as the corresponding node diagonally to the left in the bottom row.
