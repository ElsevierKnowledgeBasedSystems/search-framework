template<class MyAlgorithm, bool>
struct slb::ext::policy::backtrackLock::Copy< MyAlgorithm, bool >

A class for handling RAII for backtracking in IDA* when the parent state is kept, i.e. core::sb::StateNeighbor (i.e. not core::sb::ActionNeighbor) is used. The second template parameter is not used, just for uniformity.
