template<class Concrete, ALG_TPARAMS>
struct algorithm::Algorithm< Concrete, ALG_TPARAMS >

Common abstract base for search algorithms.

Template Parameters
logFlag_If true, the events generated by the search algorithm are logged. Otherwise, they are not.
Node_The search node type.
GoalHandlerThe policy for handling goal conditions.
HeuristicThe heuristic used by the search algorithm.
GeneratorThe generator used by the search algorithm.
